# Инструкция по запуску и проверке приложения

В текущем решении база данных SQLite **уже реализована средствами самого Android** (через встроенный класс `SQLiteOpenHelper`). Никаких сторонних библиотек для базы данных не используется — это стандартный и нативный подход в Android-разработке.

## 1. Как запустить приложение
1. Откройте проект в **Android Studio**.
2. Дождитесь окончания синхронизации Gradle (внизу справа будет ползунок, когда он исчезнет — проект готов).
3. Создайте виртуальное устройство (эмулятор) через **Device Manager** или подключите реальное устройство по кабелю/Wi-Fi.
4. Нажмите зеленую кнопку **Run** (или Shift + F10) на верхней панели Android Studio.
5. Приложение скомпилируется и запустится на экране устройства/эмулятора.

## 2. Как показать работоспособность приложения (что скриншотить)
Для отчета (например, для добавления в README.md) вам нужно сделать несколько скриншотов, которые докажут, что все требования выполнены. 

Сделайте следующие **4 скриншота**:

### Скриншот 1: Пустой список или начальный экран
- Запустите приложение.
- Сделайте скриншот главного экрана.

### Скриншот 2: Экран добавления фильма
- Нажмите на круглую кнопку с плюсиком (Floating Action Button) в правом нижнем углу.
- Откроется экран добавления. Введите данные фильма (Название, Режиссер, Год, Жанр, Стоимость).
- **Сделайте скриншот этого экрана с заполненными полями**.
- После этого нажмите кнопку "Save", чтобы сохранить фильм. Добавьте таким образом 2-3 разных фильма.

### Скриншот 3: Главный экран со списком и фильтрацией
- Вернувшись на главный экран, вы увидите список добавленных фильмов.
- Введите часть имени режиссера или названия в текстовые поля фильтров сверху.
- **Сделайте скриншот**, чтобы было видно, что список отфильтровался (показывает только подходящие фильмы).

### Скриншот 4: Демонстрация SQLite в Android Studio (Database Inspector)
Чтобы доказать, что данные реально сохраняются в SQLite, воспользуйтесь встроенным инструментом Android Studio:
1. Не закрывая запущенное на эмуляторе/устройстве приложение, в Android Studio внизу найдите вкладку **App Inspection**.
2. Откройте ее и выберите вкладку **Database Inspector**.
3. Слева в списке баз данных выберите `movies.db`, а затем таблицу `movies`.
4. В правой части появится таблица со всеми фильмами, которые вы добавили в приложении.
5. **Сделайте скриншот всего окна Android Studio**, чтобы было видно саму таблицу с данными и запущенный Database Inspector.

## 3. Куда вставить скриншоты
После того как вы сделаете эти скриншоты, сохраните их в папку `screenshots/` в корне проекта (я уже создал пустые файлы-заглушки) и замените их на свои реальные изображения. Затем они автоматически появятся в файле `README.md`.

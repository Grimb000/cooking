# Cinema App

This is a mobile application for managing a list of movies, developed in Kotlin using Jetpack Compose and SQLite.

## Features
- **Movie Catalog**: View a list of movies (id, title, director, year, genre, cost).
- **Search & Filter**: Filter movies by Title or Director.
- **Add New Movie**: Add a new movie to the SQLite database via a separate screen.
- **Jetpack Compose**: The UI is built entirely using Jetpack Compose.
- **SQLite Database**: Demonstrates local data storage using SQLiteOpenHelper.
